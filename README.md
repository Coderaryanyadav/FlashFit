# FlashFit - Ultra-Fast Fashion Delivery Platform

A modern e-commerce platform with 60-minute delivery, built with Next.js, Firebase, and TypeScript.

## Quick Start

```bash
npm install
npm run dev
```

## Apps
- **Customer App** - Main shopping experience
- **Admin Dashboard** - Order & inventory management  
- **Driver App** - Delivery tracking
- **Seller App** - Vendor portal

## Environment Variables Required

See `.env.example` for required variables.

**IMPORTANT:** After adding environment variables in Vercel, you must **redeploy** for them to take effect.
